# Affiliate MVP Laptop

Du an MVP rieng cho he thong affiliate chay tren 1 laptop server, tai nhe, xu ly theo dot.

## Muc tieu giai doan 1

- Lay san pham theo dot tu affiliate API
- Tao affiliate link
- Sinh noi dung qua AI API
- Duyet noi dung thu cong
- Dang Fanpage so luong nho
- Theo doi tien do va nhat ky job

## Cau truc

- `backend/`: PHP dashboard va API noi bo
- `workers/`: worker script cho cac job theo dot
- `docs/`: ke hoach va tracker
- `storage/`: log va du lieu tam

## Trang thai

Dang o giai doan scaffold nen tang.
